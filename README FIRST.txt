Before running the system first you need to have:
1. XAMPP Control Panel installed in your computer.
2. Crystal Report for Visual Studio installed in your computer.
3. Delete the All CrystalDecisions in the Reference at Solution Explorer then
add it again. (see Instruction 1)
4. Imported the database file in phpmyadmin. (see Instruction 2)

Intruction 1:
* Right click at References then click add References then search CrystalDecisions 
then select:
- CrystalDecisions.CrystalReports.Engine
- CrystalDecisions.ReportSource
- CrystalDecisions.Shared
- CrystalDecisions.Windows.Forms

Instruction 2:
* Go to phpmyadmin then select import tab then click Choose File, 
then go to database folder at the Payroll-System folder and select payroll_system_db then
click the Go button.

Enjoy!
